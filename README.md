## Overview

**nwkit** is a pre- and post-processing tool for phylogenetic trees. 

## Dependency
* [python3]()
* [ete3]()

## Installation
```
# Installation with pip
pip install git+https://github.com/kfuku52/nwkit

# This should show complete options
nwkit -h 
```
